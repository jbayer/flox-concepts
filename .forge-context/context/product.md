# Product Context

## What This Project Does

A collection of markdown files that explains Flox incrementally
in simple, plain language. Educational documentation designed
to onboard readers progressively.

## Architecture

Documentation / knowledge base — markdown files only, no code.

## Content Structure

The documentation is organized in three layers:

1. **Principles** — foundational "why" concepts, not tied to
   any specific technology
2. **Concepts** — high-level vocabulary and mental models
3. **Flox Specifics** — concrete examples using Flox that
   illustrate the concepts

## Key Dependencies

None — plain markdown, no build tools or frameworks.

## Deployment

Static markdown files, readable directly on GitHub or any
markdown renderer.
