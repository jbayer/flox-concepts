# Checklist: Define Principles

Epic health and progress tracking. Unlike stories,
epics don't have a fixed end state — this checklist
tracks milestones and ongoing health.

---

## Initial Setup

- [x] Epic directory created
- [x] Problem statement drafted in effort.md
- [x] Initial stakeholders identified

## Problem Framing

- [x] Problem space clearly defined
- [x] "Why now?" articulated
- [x] Current state documented
- [x] Key stakeholders identified with needs/impact

## Story Refinement

- [x] Initial user stories captured
- [x] At least one story refined to "Ready" status
- [x] Acceptance criteria defined for ready stories

## Scope Identification

- [x] First story candidate identified
- [x] Acceptance criteria clear and testable

## Story Spawning

- [x] Candidate approved
- [ ] Story spawned from candidate
